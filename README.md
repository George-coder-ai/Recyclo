# Recyclo Mobile Application

This is a full-stack Recyclo application that allows users to submit, track, and find recyclable materials.

## Project Structure
- **frontend/**: Contains React Native application files.
- **backend/**: Contains Node.js API server files.
- **database/**: Contains database schema files.

## Getting Started

### Frontend
1. Navigate to the `frontend` directory.
2. Run `npm install` to install dependencies.
3. Start the application with `npm start`.

### Backend
1. Navigate to the `backend` directory.
2. Create a `.env` file with necessary environment variables.
3. Run `npm install` to install dependencies.
4. Start the server with `node server.js`.
